# React + Tailwind Starter (Vite)

A minimal React (Vite) project with Tailwind CSS already configured.

## Quick Start

```bash
npm install
npm run dev
```

Then open the printed local URL in your browser.

## Build for production

```bash
npm run build
npm run preview
```

## What's included
- Vite + React
- Tailwind CSS 3
- PostCSS + Autoprefixer
- Ready-to-edit `App.jsx`

## Tailwind config
Tailwind scans `index.html` and everything in `src/**/*.{js,jsx,ts,tsx}`.
